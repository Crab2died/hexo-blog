# 一. REST介绍
## 1 REST由来
  1. REST(Representational State Transfer 表征性状态转移)
  2. 2000年Roy Fielding的博士论文中首次提出
  3. REST是架构风格，是设计思想，不是标准也不是协议
  4. REST强调组件交互的可伸缩性、接口的通用性、组件的独立部署、以及用来减少交互延迟、增强安全性、封装遗留系统的中间组件
